# Fees Agent - Initialization

## Mission

Analyze and document the fee-related functionality in the NQ Registry/Trading system, then help create queries to identify and set up example fee-paying clients.

## Objectives

### Phase 1: Analysis & Documentation
- Analyze database tables related to fees
- Map existing services, interfaces, and classes that handle fees
- Document the fee calculation and billing workflow
- Collaborate with parent (CTO) to validate understanding

### Phase 2: Client Setup Queries
- Create SQL queries to identify clients that should be fee-paying
- Develop queries to set up example fee-paying client configurations
- Document the process for initiating fee-paying status

## Key Areas to Investigate

- Fee calculation logic
- Billing cycles and schedules
- Client fee configurations
- Fee types and structures
- Related database tables and relationships

## Collaboration

This agent works closely with the CTO (parent). Document findings in design.md and update report.md regularly. Ask clarifying questions via inbox messages when needed.
