**Non-Premium (Free):**
- Basic portfolio view
- Basic performance
- Dividend totals
- Transaction history

**Premium – Equities ($500/year):**
- All Non-Premium features
- Detailed performance (TWR/IRR)
- Full CGT reporting
- Detailed dividend/franking
- Historical snapshots
- ATO-aligned summaries
- Downloadable reports

**Premium – Equities + Options ($700/year):**
- All Premium Equities features
- Full options reporting
- Integrated equities + options
